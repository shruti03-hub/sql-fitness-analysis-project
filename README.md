# sql-fitness-analysis-project
A simple yet powerful SQL project designed to track users, exercises, and workout sessions — built to practice database design, query writing, and real-world data analysis.
Project: Fitness Workout Database Analysis (SQL + MySQL)

About the Project
A SQL-based analytics project to explore workout patterns, calories burned, and user activity using relational tables.

Files Included:
1)schema.sql – all table creation scripts
2)insert_data.sql – sample dataset
3)queries.sql – 10 business-focused SQL queries
4)results.png – screenshots of output 

Tools Used:
1)MySQL Workbench
2)SQL

Key Queries:
Total users
Top 3 calorie-burning users
Users who performed specific exercises
Youngest & oldest user
Workout duration insights

Learnings:
-Joins, Aggregations, Subqueries
-Schema design & relationships
-Real-world analytics mindset
